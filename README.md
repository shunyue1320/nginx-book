## 安装

如果你没有安装 gitbook，则先使用 npm（或者 cnpm）安装 `gitbook-cli`：

```bash
npm install gitbook-cli -g
```

进入项目根目录，执行以下命令运行 gitbook：

**安装依赖**

```bash
gitbook install
```

**启动本地服务器**

```bash
gitbook serve
```

**生成 HTML 文件，存放在 `_book` 目录下**

```
gitbook build
```
